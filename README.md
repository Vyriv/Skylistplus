# **Skylist+**

Skylist+ is an expansion mod for Skylist that adds advanced player tracking, party control, and list management tools.

This module focuses on improving party quality and giving you full control over who you play with.

Skylist+ requires Skylist to be installed.

## **Features**

* Local thrower list with custom reasons and expiry
* Shared global list sync (M5 community list)
* Real-time detection of listed players in parties
* Automatic party removal (autokick) with leader checks
* Party chat alerts with reasons and appeal info
* Clickable client-side notifications for manual control
* In-game list browser with search and filtering

## **Party Integration**

* Autokick only runs when you are party leader
* Detects joins via chat in real time
* Announces flagged players before taking action
* Toggle between automatic and manual responses

## **Commands**

* `/tl`, `/thrower`, `/throwerlist` — Open list UI
* `!tlcheck <ign>` — Check players from party chat

## **Dependencies**

* Skylist
* Fabric API
* Fabric Language Kotlin
